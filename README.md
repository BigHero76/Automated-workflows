# Research Digest System

AI-powered research paper digest dashboard.

## Features
- Displays summarized research papers
- Search functionality
- Designed to connect with n8n backend

## Future Plans
- Automated paper ingestion
- AI summaries
- Personalized feeds


## Architecture

Frontend → n8n Webhook → AI Summarization → Database → Dashboard

## Tech Stack
- n8n (automation backend)
- JavaScript frontend
- AI summarization (LLM API)
- GitHub Pages hosting
